# Задания для интервью QA Automation на Python.

## Описание

В данном репозитории собраны задания для интервью на позицию QA Automation на языке Python.
Помимо самих заданий здесь также описаны ответы как в конечном виде, так и в виде направлений мышления потенциального 
кандидата на вакансию.

Идея создания такой обновляемой базы знаний родилась из потребности более тщательной проверки кандидатов на вакансию. 
Здесь нет стандартных вопросов из различных статей и чек-листов, которые помогают начинающим автоматизаторам
тестирования пройти быстрый отбор у HR-специалистов. Напротив, часто будут попадаться вопросы, которые не предполагают
точного ответа, а заставляют проявить фантазию, и тем самым дают понимание об опыте специалиста в каком-либо направлении.

Вопросы касаются общего стека автоматизации на Python: 
* PyTest
* Requests
* Playwright
* Pydantic
* Loguru
* GitLab CI
* Docker
* Allure Framework
